# Documentation Layout

- `reports/` contains project status, compliance, evaluation, implementation, and presentation notes.
- `kiosk-design/` contains source design prototypes and design-token references used to build the current kiosk UI.
- `SUVIDHA_ENTERPRISE_ARCHITECTURE.md` and `secure-upload-flow.txt` describe architecture and upload flow.

The kiosk runtime does not depend on this folder directly.
